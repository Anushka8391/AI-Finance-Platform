# 💰 Welth – AI-Powered Finance Platform

## 🧠 Overview

Welth is an AI-powered financial management platform that helps users track expenses, analyze spending behavior, and generate intelligent insights for better financial decisions.

The system integrates real-time data processing with AI-driven analysis to provide actionable recommendations and improve financial awareness.

---

## 🤖 AI Features

* Automated financial insights based on transaction data
* Intelligent categorization of expenses
* Predictive analysis of spending patterns
* Real-time AI-generated recommendations

---

## ⚙️ Tech Stack

* **Frontend:** Next.js, Tailwind CSS, Shadcn UI
* **Backend:** Node.js (API Routes)
* **Database:** Supabase, Prisma
* **AI Integration:** Gemini API / Custom logic
* **Background Jobs:** Inngest
* **Security:** Clerk Authentication
* **Deployment:** Vercel

---

## ✨ Key Features

* 📊 Real-time expense tracking
* 🤖 AI-generated financial insights
* 🔐 Secure authentication system
* ⚡ Fast and responsive UI
* 📱 Mobile + desktop compatibility
* 🔄 Background job processing

---

## 📊 Project Highlights

* Handles **real-time financial data processing**
* Generates insights with **~1–2 second latency**
* Built using **10+ reusable components**

---

## 🧪 System Architecture

* Frontend interacts with backend via API routes
* Backend processes financial data and triggers AI insights
* Background jobs handle asynchronous tasks
* Database stores and manages user financial data
